## 📖 Description

This video demonstrates the security assessment of a Node.js-based web application performed as part of Week 1 of the cybersecurity internship task.

---

## 🧪 What is Covered in the Video

* Application setup and local deployment
* Exploration of key features (Signup, Login, Dashboard)
* Manual security testing:

  * Cross-Site Scripting (XSS)
  * SQL Injection
  * Input validation testing
* Automated vulnerability scanning using OWASP ZAP
* Identification of security misconfigurations
* Explanation of key vulnerabilities such as brute-force risk

---

## 🛠️ Tools Used

* OWASP ZAP
* Browser Developer Tools
* Node.js Application

---

## 🎯 Key Outcome

The assessment helped identify:

* Security misconfigurations
* Lack of brute-force protection
* Areas for improving application security

This video is part of the internship submission and demonstrates practical application of cybersecurity concepts through hands-on testing.
